# Project brief

A fast, mobile‑first one‑page site that funnels visitors to **Order on Uber Eats**, **Order on DoorDash**, **View Menu**, and **Call/Catering**. Keep copy short, images optimized, and buttons thumb‑friendly.

---

## File tree (suggested)

```
/ (root)
  public/
    images/hero.jpg
    images/gallery-*.jpg
  src/
    app.tsx              # one-page React component (static export OK)
    menu.json            # editable menu content
  vercel.json            # SPA rewrites if you deploy to Vercel
  README.md
```

## Implementation notes

* Replace hero and gallery images with your own optimized WebP/JPEG (1200px wide for hero, 800px for cards).
* Track button clicks with GA4 events: `order_ubereats_click`, `order_doordash_click`.
* Add JSON‑LD schema (`Restaurant`) with name, address, phone, hours, `hasMenu` (link to `/menu.json`), and delivery links under `makesOffer`.
* Keep site copy short; avoid duplicate text from Zawadi Center to help SEO.
* When DoorDash goes live, set the DoorDash button to active and add UTM tags.
